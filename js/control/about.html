Civic Graph is a crowdsourced knowledge base that maps people and organizations doing civic tech work in communities around the globe. We define 'civic tech' as the use of technology for public good. Use Civic Graph to:

  1. Familiarize yourself with the civic tech space
  2. Build new collaborations, identify funding opportunities, pursue data exchanges, and more

Civic Graph is an open source project by Microsoft licensed under a CC BY-SA 4.0. You can contribute to or borrow from the source code here. Be sure to:

  1. Add yourself to the graph
  2. Add an organization or person who works in civic tech
  3. Share Civic Graph
