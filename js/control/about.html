Civic Graph is a crowdsourced knowledge base that maps people & organizations doing civic tech work in communities around the globe. We define 'civic tech' as the use of technology for the public good. Civic Graph can be used to:

  1. Familiarize you with the civic tech space.
  2. Enable you to use the graph as a means for a particular end (e.g. new collaborations, funding opportunities, data exchanges, etc.)

Civic Graph is an educational tool, an open information platform, and a resource for network transparency. Civic Graph is an open source project by Microsoft licensed under a CC BY-SA 4.0. Feel free to contribute to or borrow from the source code here.

Help build the most robust database of people and organizations working in civic tech:

  1. Add yourself to the graph
  2. Add an organization or person who works in civic tech
  3. Share Civic Graph
