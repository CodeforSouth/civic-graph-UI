Civic Graph is a crowdsourced knowledge graph that maps people & organizations doing work in civic tech across all six inhabited continents. We define 'civic tech' as the use of technology for the public good. Civic Graph has two main use cases:

  1. To familiarize yourself with the civic tech space.
  2. To use the tool as a means for a particular end (collaborations, funding, data exchanges, etc.)

It is an educational tool, an open information platform, and a resource for network transparency. Civic Graph is an open source project by Microsoft licensed under a CC BY-SA 4.0. Take a look at the source code here.

You can help us build the most robust database of people and organizations working in civic tech:

  1. Add yourself to the graph
  2. Add an organization or person who works in the civic tech space
  3. Share Civic Graph
